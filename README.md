🏥 MediNova

A modern, responsive healthcare web application built with HTML, CSS, and JavaScript.

📋 Overview

MediNova provides users with a clean, intuitive platform for healthcare information and services. Built entirely with vanilla HTML, CSS, and JavaScript, it delivers a lightweight, fast, and accessible experience without relying on heavy frameworks.

Whether you're a patient exploring healthcare options or just curious about medical information, MediNova makes navigation simple and enjoyable.

✨ Features

🖥️ Responsive Design – Optimized for mobile and desktop screens.

🎨 Modern UI – Professional, clean aesthetic tailored for healthcare.

⚡ Lightweight – Pure HTML, CSS, and JavaScript with no heavy dependencies.

🧭 Intuitive Navigation – Easy-to-use interface for patients and healthcare seekers.

🚀 Getting Started
Prerequisites

All you need is a modern web browser. No installations or build tools required.

Running Locally

Clone the repository

git clone https://github.com/palakkumari24/medinova.git

Navigate into the project folder

cd medinova/medinova

Open in your browser

Simply open index.html in your browser, or serve it with a local server:

npx serve .
# or
python -m http.server 8000

Visit http://localhost:8000
 to view the app.
