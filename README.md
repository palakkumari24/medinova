🏥 MediNova
A modern healthcare web application built with HTML, CSS, and JavaScript.

HTML CSS JavaScript

📋 Overview
MediNova is a responsive, frontend healthcare web application designed to provide users with a clean and intuitive medical platform experience. Built entirely with vanilla HTML, CSS, and JavaScript — no frameworks required.

✨ Features
🖥️ Responsive Design — Fully mobile-friendly and adapts to all screen sizes
🎨 Modern UI — Clean, professional aesthetic tailored for a medical/healthcare context
⚡ Lightweight — Pure HTML/CSS/JS with no heavy dependencies
🧭 Intuitive Navigation — Easy-to-use interface for patients and healthcare seekers
🚀 Getting Started
Prerequisites
All you need is a modern web browser. No installations or build tools required.

Running Locally
Clone the repository

git clone https://github.com/palakkumari24/medinova.git
Navigate into the project folder

cd medinova/medinova
Open in your browser

# Simply open index.html in your browser, or use a local server:
npx serve .
# or
python -m http.server 8000
Visit http://localhost:8000 in your browser.
