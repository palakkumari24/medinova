# Medinova.ai - React Application

This is the React version of the Medinova.ai healthcare platform.

## Installation

```bash
npm install
```

## Running the Application

```bash
npm start
```

The app will run on http://localhost:3000

## Building for Production

```bash
npm run build
```

## Project Structure

- `src/` - React source code
  - `components/` - Reusable components (Header, Footer, etc.)
  - `pages/` - Page components (Home, Questions, VoiceInput, etc.)
  - `App.js` - Main app component with routing
  - `index.js` - Entry point
- `public/` - Static files
- CSS files are kept in `src/` directory and imported in components

## Features

- React Router for navigation
- All HTML pages converted to React components
- CSS files preserved and imported
- JavaScript functionality maintained separately
